# Pathfinding Visualizer

A Google Maps inspired React application for visualizing pathfinding algorithms. Built for Design and Analysis of Algorithms (DAA) study and demonstration.

## Features
- **Algorithms:** Dijkstra, A* Search, Breadth-First Search (BFS), Depth-First Search (DFS).
- **Interactive Grid:** Drag and drop start/end nodes, draw walls/obstacles to see how the algorithms adapt.
- **Modern UI:** Clean, soft aesthetic inspired by Google Maps and modern navigation software.
- **High Performance:** Direct DOM manipulation for smooth animations without taxing the React rendering cycle.

## Tech Stack
- React (TypeScript)
- Vite
- Tailwind CSS v4
- Lucide React (Icons)

## Setup Instructions

1. **Install Dependencies**
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

2. **Run the Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to the URL provided by Vite (typically `http://localhost:5173`).
